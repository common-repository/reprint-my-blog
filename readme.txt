== Description ==

This plugin help you to get the updates of writers' other blogs by checking feeds.
Each writer has a particular configuration.
When updates are detected, the writer can decide how to show the posts.
Writers can also decide whether to redirect to the original post.
Detailed configurations are provided.
This plugin is compatible with Configure SMTP.
WARNING: use this plugin ONLY when you are sure it is used LEGALLY.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
